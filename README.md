# Monster Rolodex React App

This app created with `create-react-app`

## Run Locally

Clone the project

```bash
  git clone https://github.com/bilgihankose/react-monster-rolodex.git
```

Go to the project directory

```bash
  cd react-monster-rolodex
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
